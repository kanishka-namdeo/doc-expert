# AI Interaction Components

## Purpose

React components for AI-powered features: citations, reasoning display, and source references.

## Ownership

- Citation rendering components
- AI reasoning/thinking display
- Source document references
- Chat message components

## Local Contracts

- Client components use `"use client"` directive
- Components handle streaming AI responses
- Use `cn()` utility for conditional class merging

## Work Guidance

- Components render AI output with proper formatting
- Citations link back to source documents
- Reasoning display shows AI thinking process when available
